# Retail Sales Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes retail sales performance using the Superstore dataset.

## Key Metrics
- Monthly Revenue Trend
- Profit Margin by Product Category
- Revenue by Region
- Loss Order Rate %

## Tools Used
- Power BI
- DAX
- Data Visualization
- Excel

## Dashboard Preview

![Dashboard](dashboard_screenshot.png)

## Insights
- Technology has the highest profit margin.
- West region generates the highest revenue.
- Revenue peaks during the final quarter.
- Loss Order Rate is 18.72%.
